# Agent Guidance

## Developer Commands
- `npm run dev`: Start development server
- `npm run build`: Run typecheck (`tsc -b`) and build
- `npm run lint`: Run ESLint
- `npm run preview`: Preview production build

## Verification Flow
`npm run lint` -> `npm run build` (includes typecheck)

## Tech Stack
- React 19 + TypeScript
- Vite (Build tool)
- Tailwind CSS 4
- Framer Motion (Animations)
- Radix UI (Primitives)
- Lucide React (Icons)
